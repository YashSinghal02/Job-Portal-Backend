# Job Portal Backend

Backend API for a Job Portal application where employers can post jobs and employees can apply or save jobs.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Features
- User Authentication
- Employer can create jobs
- Employees can apply for jobs
- Save jobs functionality
- REST API structure
- Error handling middleware

## Folder Structure
src/
Controllers/
Database/
Middlewares/
Model/
Routes/
Service/
Templates/
Utils/

## Installation

1. Clone the repository
git clone https://github.com/YashSinghal02/Job-Portal-Backend.git

2. Install dependencies
npm install

3. Create .env file

4. Run server
npm start
